# Acheto E-commerce

A modern e-commerce platform built with Next.js, TailwindCSS, Flask, and MongoDB.

## Tech Stack

- **Frontend**: Next.js 14, TailwindCSS, TypeScript
- **Backend**: Flask (Python), Flask-CORS
- **Database**: MongoDB
- **Icons**: Custom SVG icons

## Prerequisites

- Node.js 18+
- Python 3.9+
- Docker & Docker Compose (for MongoDB)
